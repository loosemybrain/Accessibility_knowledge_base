#### Dokumentierte Herstellerinformation

SAP dokumentiert, dass für den Betrieb von SAP GUI for Windows im Accessibility Mode zusammen mit dem Screenreader Freedom Scientific JAWS die SAP Screen Reader Extensions installiert werden müssen.

SAP dokumentiert die Komponente unter der Bezeichnung `SAP GUI Screen Reader Extension for JAWS` mit dem technischen Komponentenkürzel `SRX`.

SAP dokumentiert, dass SAP GUI Screen Reader Extensions die Nutzung von SAP GUI zusammen mit dem Screenreader JAWS for Windows ermöglichen.

SAP dokumentiert, dass die Extensions JAWS-basierte Skripte sind, die SAP GUI über das SAP-GUI-Objektmodell zugänglich machen.

SAP dokumentiert als Voraussetzungen für SRX:

- Freedom Scientific JAWS ist in einer unterstützten Version installiert.
- SAP GUI ist installiert oder wird installiert.

SAP dokumentiert zusätzlich eine Komponente `Accessibility settings for SAP GUI`. Diese Komponente wendet typische Einstellungen an, die für die Nutzung von SAP GUI for Windows zusammen mit einem Screenreader erforderlich sind. SAP nennt als eine dieser Einstellungen die standardmäßige Aktivierung des SAP GUI Accessibility Mode.

SAP dokumentiert, dass unbeaufsichtigte Installationen mit automatischer Patch-Installation auf dem Client durchgeführt werden können, indem die entsprechende Kommandozeile in das Anmeldeskript des Clients eingetragen wird.

SAP dokumentiert alternativ SAP Automatic Workstation Update.

SAP dokumentiert, dass SAP Automatic Workstation Update Workstations aktualisiert, wenn der Installation Server gepatcht wurde oder installierte Pakete aktualisiert wurden.

SAP dokumentiert, dass SAP Automatic Workstation Update Workstations bei Bedarf neu startet.

SAP dokumentiert für die Aktualisierung des Frontends:

- Update des Frontends von einem Installation Server.
- Patchen eines Standalone Frontends.

SAP dokumentiert, dass ein Update dieselben Schritte wie eine Installation umfasst.

SAP dokumentiert für ein Upgrade auf eine neue Hauptversion, dass die SAP-GUI-Version vom Presentation-DVD-Installationsmedium verwendet werden muss und danach Patches zur Aktualisierung der neuen SAP-GUI-Version genutzt werden können.

SAP dokumentiert separate Patches für `Screen Reader Extension`.

SAP dokumentiert für die Deinstallation des SAP Front End:

- SAP GUI for Windows beziehungsweise SAP Business Client dürfen zum Deinstallationszeitpunkt nicht laufen.
- Wenn eine Anwendung während der Deinstallation läuft, wird die Benutzerin oder der Benutzer anschließend zu einem Neustart aufgefordert.
- Die Deinstallation kann über die installierte Anwendung im Windows-Systemdialog zum Entfernen von Programmen durchgeführt werden.

SRE-spezifische administrative Registry-Einstellungen sind in den verwendeten Primärquellen nicht dokumentiert.
