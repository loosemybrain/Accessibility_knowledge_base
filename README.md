## Zweck

Dieses Projekt enthält die Arbeitsgrundlage für die Accessibility Knowledge Base des ITZBund.

Die Dokumentation wird als fachlich belastbare, nachvollziehbare und langfristig wartbare Wissensplattform zur digitalen Barrierefreiheit aufgebaut.

## Arbeitsprinzip

Fachliche Aussagen werden erst veröffentlicht, wenn sie durch Primärquellen, nachvollziehbare Evidenzen oder reproduzierbare Praxistests abgesichert sind.

Die technische Validierung erfolgt über:

```bash
npm run akb:validate
```
