#### Dokumentierte Herstellerinformation

SAP dokumentiert für SAP GUI for Windows Hinweise für Benutzerinnen und Benutzer von Screenreadern und Bildschirmlupen.

SAP dokumentiert eine Übersicht von SAP-GUI-Einstellungen für Screenreader und Bildschirmlupen.

SAP dokumentiert für SAP GUI for Windows den Einsatz des Screenreaders Freedom Scientific JAWS im Zusammenhang mit dem Accessibility Mode.

SAP dokumentiert, dass für den Betrieb von SAP GUI for Windows im Accessibility Mode zusammen mit dem Screenreader Freedom Scientific JAWS die SAP Screen Reader Extensions installiert werden müssen.

SAP dokumentiert die SAP Screen Reader Extensions als Komponente `SAP GUI Screen Reader Extension for JAWS` mit dem technischen Komponentenkürzel `SRX`.

SAP dokumentiert, dass SAP GUI Screen Reader Extensions die Nutzung von SAP GUI zusammen mit JAWS for Windows ermöglichen.

SAP dokumentiert als Voraussetzungen für SRX:

- Freedom Scientific JAWS ist in einer unterstützten Version installiert.
- SAP GUI ist installiert oder wird installiert.

#### Dokumentierte Herstellerinformation

SAP dokumentiert für SAP GUI for Windows die Verwendung des Screenreaders Freedom Scientific JAWS zusammen mit dem Accessibility Mode.

SAP dokumentiert, dass für den Betrieb von SAP GUI for Windows im Accessibility Mode zusammen mit Freedom Scientific JAWS die SAP Screen Reader Extensions installiert sein müssen.

SAP dokumentiert folgende Zusammenhänge und Voraussetzungen:

- Die SAP GUI Screen Reader Extensions ermöglichen die Nutzung von SAP GUI zusammen mit JAWS for Windows.
- Die Extensions sind JAWS-basierte Skripte, die SAP GUI über das SAP-GUI-Objektmodell zugänglich machen.
- Freedom Scientific JAWS muss in einer unterstützten Version installiert sein.
- SAP GUI muss installiert sein oder installiert werden.
- Die SAP Screen Reader Extensions greifen über die SAP GUI Scripting-Schnittstelle auf die Elemente des SAP-GUI-Fensters zu.
- SAP GUI Scripting muss für die betreffenden Anwendenden mindestens als Read-Only-Variante aktiviert sein.

SAP dokumentiert für Benutzerinnen und Benutzer von Screenreadern folgende empfohlene Einstellungen:

| Einstellung | Empfehlung für Screenreader |
|-------------|-----------------------------|
| Accessibility Mode | Einschalten |
| Benachrichtigung beim Anfügen eines Skripts an eine laufende SAP GUI | Ausschalten |
| Schreibgeschützte und deaktivierte Elemente in die Tabulatorreihenfolge einbeziehen | Einschalten |
| Listensymbole als Buchstaben anzeigen | Einschalten |
| Dialogfeld für Meldungen | Einschalten |
| F4-Hilfe als modales Dialogfeld anzeigen | Einschalten |
| Office-Dokumente zur besseren Screenreader-Unterstützung extern öffnen | Einschalten |

SAP dokumentiert als geeignete Themes für die Verwendung mit Screenreadern und Bildschirmlupen Quartz, Belize, SAP Signature und Classic. Zusätzlich nennt SAP Quartz High Contrast Black, Quartz High Contrast White, Belize High Contrast Black und Belize High Contrast White.

SAP dokumentiert folgende Hinweise und Einschränkungen im Zusammenhang mit JAWS:

- Beim Zugriff auf den Optionen-Dialog mit JAWS wird automatisch eine vereinfachte Baumdarstellung verwendet. Dadurch kann der Accessibility Mode mit JAWS in jedem verfügbaren Theme aktiviert werden.
- Die Suchfunktion des Optionen-Dialogs steht in dieser vereinfachten Darstellung aus technischen Gründen nicht zur Verfügung.
- Die Einstellung zur Benachrichtigung beim Anfügen eines Skripts soll bei der gemeinsamen Nutzung von Accessibility Mode und JAWS deaktiviert werden. Andernfalls wird jedes Mal ein Benachrichtigungsdialog angezeigt, wenn JAWS auf Objekte innerhalb von SAP GUI zugreift.

#### Dokumentierte Herstellerinformation

Die ausgewerteten offiziellen SAP-Quellen dokumentieren allgemeine Funktionen und empfohlene Einstellungen für Benutzerinnen und Benutzer von Screenreadern. In diesen allgemeinen Angaben wird NVDA nicht ausdrücklich genannt.

SAP verweist für Frontend-Anforderungen, Infrastruktur, besondere Einstellungen und Informationen zu den von SAP GUI unterstützten Screenreadern auf SAP Note 1139953. Der Inhalt dieses SAP-Hinweises wurde im Rahmen dieser Recherche nicht verifiziert. Daher kann aus diesem Verweis keine Aussage zur Unterstützung von NVDA übernommen werden.

Die ausgewertete Komponentenliste für SAP GUI for Windows dokumentiert die SAP GUI Screen Reader Extensions ausdrücklich für JAWS. Sie enthält keine ausdrückliche Aussage zu NVDA.

In den verwendeten offiziellen SAP-Quellen wurden keine eindeutigen Herstellerinformationen zu folgenden NVDA-spezifischen Punkten gefunden:

- offizielle Unterstützung von NVDA,
- Voraussetzungen für den Einsatz von NVDA,
- NVDA-spezifische Konfiguration,
- Abhängigkeiten für NVDA,
- Zusammenhang zwischen NVDA und dem Accessibility Mode,
- Zusammenhang zwischen NVDA und den SAP Screen Reader Extensions,
- bekannte NVDA-spezifische Hinweise oder Einschränkungen,
- unterstützte NVDA-Versionen.

Aus den dokumentierten Angaben zu JAWS wird keine Aussage zu NVDA abgeleitet.
