#### Dokumentierte Herstellerinformation

SAP dokumentiert für SAP GUI for Windows Hinweise für Benutzerinnen und Benutzer von Screenreadern und Bildschirmlupen.

SAP dokumentiert eine Übersicht von SAP-GUI-Einstellungen für Screenreader und Bildschirmlupen.

SAP dokumentiert für SAP GUI for Windows den Einsatz des Screenreaders Freedom Scientific JAWS im Zusammenhang mit dem Accessibility Mode.

SAP dokumentiert, dass für den Betrieb von SAP GUI for Windows im Accessibility Mode zusammen mit dem Screenreader Freedom Scientific JAWS die SAP Screen Reader Extensions installiert werden müssen.

SAP dokumentiert die SAP Screen Reader Extensions als Komponente `SAP GUI Screen Reader Extension for JAWS` mit dem technischen Komponentenkürzel `SRX`.

SAP dokumentiert, dass SAP GUI Screen Reader Extensions die Nutzung von SAP GUI zusammen mit JAWS for Windows ermöglichen.

SAP dokumentiert als Voraussetzungen für SRX:

- Freedom Scientific JAWS ist in einer unterstützten Version installiert.
- SAP GUI ist installiert oder wird installiert.
